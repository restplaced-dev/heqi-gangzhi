# 河憩缸誌 MVP 產品規格

## 一句話定位

記錄魚缸日常，把魚當寵物照顧。

## 目標使用者

- 一般水族飼主
- 多缸玩家
- 想記錄魚隻狀態的人
- 想分享魚缸日常的人
- 不一定懂專業水質術語的新手

## 使用者痛點

1. 忘記上次何時換水、餵食、清潔。
2. 魚狀態變差時，想不起前幾天做了什麼。
3. 魚隻不像貓狗一樣容易被當成個體記錄。
4. 一般水族工具太硬派，像表格或水質計算器。
5. 社群貼文零散，沒有和自己的魚缸紀錄連起來。

## MVP 核心功能

### 魚缸

- 建立魚缸
- 記錄尺寸 / 類型 / 狀態
- 顯示魚隻檔案
- 顯示最近紀錄

### 魚隻

- 名字
- 魚種
- 個性 / 習慣
- 喜歡的食物
- 常待位置
- 入缸日期
- 照片時間軸，未來功能

### 日常紀錄

- 觀察
- 餵食
- 換水
- 拍照
- 清潔
- 新魚入缸
- 狀態怪怪的

備註可留空。

### 提醒

- 換水
- 餵食
- 拍照
- 清潔
- 觀察

提醒可依魚缸個別設定。

### 社群

- 魚缸動態牆
- 讚
- 留言，原型可先假資料
- 收藏，原型可先 UI
- 標籤
- 可由紀錄同步成貼文

## 不做事項

- 不做用藥建議
- 不做疾病診斷
- 不做繁殖主線
- 不做電商
- 不做真正付款
- 不做真正醫療諮詢

## 未來 Pro 方向

- 不限魚缸
- 不限魚隻
- 完整照片時間軸
- 雲端同步
- 自訂提醒
- 月報
- 匯出魚缸紀錄
- 貼文模板
- 私人日記
