# 河憩缸誌｜養魚日常紀錄 App 原型

這是一個可在手機瀏覽器測試的 PWA 靜態原型，重點是測試流程與產品感：

- 首頁：今日魚缸、提醒、最近紀錄
- 魚缸：魚缸檔案、魚隻個體檔案、新增魚缸
- 紀錄：觀察、餵食、換水、拍照、清潔、新魚入缸、狀態怪怪的
- 社群：魚缸動態、按讚、貼文標籤
- 我的：訂閱方向與資料重置

## 測試方式

直接開啟 `index.html` 即可測試。資料會存在瀏覽器 localStorage。

若要放到 GitHub Pages：
1. 建立一個 repo。
2. 上傳這個資料夾裡的所有檔案。
3. 到 Settings → Pages → Deploy from branch。
4. 選 main branch / root。

## 目前刻意不做

- 先聚焦日常紀錄與社群互動
- 不把 App 做成醫療型工具
- 不把繁殖當主線
- 不做付款與登入
- 不做真正雲端社群

這版主要用來測試：飼主是否願意每天打開、紀錄、發魚缸日常。
